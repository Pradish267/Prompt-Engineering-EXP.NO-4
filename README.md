Ex.No.4-Generate the Prompt for the following prompt types - Comparative Analysis Prompt Experiential Perspective Prompt - Everyday Functioning Prompts.

Aim: 
To write the prompt for the following prompt types. 
1. Comparative Analysis Prompt 

2. Experiential Perspective Prompt
   
3. Everyday Functioning Prompts 

Explanation -  Write the prompt for any one use case from Unit 5 or any 2 own use cases for each type.

Procedure:

1.Define the Scenario and Use Case: 

Scenario:
You are planning to buy a new smartphone for your daily use. You want a phone that supports long battery life, a good AI-powered camera for photography, and smooth performance for everyday tasks such as social media, online classes, and entertainment. However, you are confused between two smartphone models and want to make the best choice. At the same time, you also want to understand how such a smartphone would help you in your daily life.


Compare the two smartphones in terms of battery life, AI camera features, performance, and price. Then explain from the perspective of a regular user how these features improve everyday activities such as communication, photography, and productivity, and suggest which smartphone would be more suitable for daily use.

2.Identify Prompt Patterns for Each Aspect: Idea Generation Prompts:
Refer the following example for each prompt type:

Comparative Analysis Prompt : 
  • "Compare and contrast deep learning and machine learning."
  • "How do renewable and non-renewable energy sources differ?"

Experiential Perspective Prompt:

  “As a software engineer, what are the benefits of using cloud computing in your projects?” 
  
  “From a student’s perspective, describe the challenges of online learning.”

Everyday Functioning Prompts:

  “Describe the role of machine learning in everyday apps like YouTube or Netflix."
 
  “How is encryption used in daily life when sending WhatsApp messages?”

  OUTPUT:
  1. Comparative Analysis Prompts
Use Case 1: Smartphone Comparison

Prompt:

Compare and contrast the Samsung Galaxy A55 and iPhone 15 in terms of battery life, AI camera features, performance, storage, and price. Analyze their advantages and disadvantages for students who use smartphones for online classes, social media, photography, and entertainment. Recommend the better option for daily use with reasons.

Use Case 2: Electric Vehicle vs Petrol Vehicle

Prompt:

Compare and contrast electric vehicles and petrol vehicles based on cost, maintenance, environmental impact, performance, and convenience. Analyze the benefits and limitations of each option and recommend which is more suitable for daily commuting.

2. Experiential Perspective Prompts
Use Case 1: Student Using a Smartphone

Prompt:

As a college student, describe your experience using a smartphone with a long-lasting battery, AI-powered camera, and smooth performance. Explain how these features help in attending online classes, communicating with friends and teachers, completing assignments, and managing daily activities.

Use Case 2: Remote Employee Using Cloud Tools

Prompt:

From the perspective of a remote employee, describe your experience using cloud-based collaboration tools such as Google Drive and Microsoft Teams. Explain how these tools help you communicate with team members, share files, manage projects, and improve productivity.

3. Everyday Functioning Prompts
Use Case 1: AI in Smartphones

Prompt:

Explain how AI-powered features in smartphones, such as camera enhancements, voice assistants, and battery optimization, improve everyday activities like communication, photography, learning, and entertainment.

Use Case 2: AI in Online Platforms

Prompt:

Describe how artificial intelligence is used in everyday applications such as YouTube, Netflix, Google Maps, and online shopping websites. Explain how AI helps users receive recommendations, navigate efficiently, and improve their overall experience.

These prompts demonstrate how AI can be guided to compare alternatives, present viewpoints from specific user perspectives, and explain the role of technology in everyday life. Through this activity, the structure and application of different prompt patterns were understood and implemented effectively.
